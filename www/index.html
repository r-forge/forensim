Welcome to The forensim package project!

The forensim package for the R environment provides statistical tools dedicated to the interpretation of forensic DNA mixtures

PACKAGE SOURCES
  • Source (for Linux, Unix, or MacOS/Windows with development tools installed)
  • Windows binary
  • MacOS X binary

The development  version  is  available from R-Forge's daily snapshots:
  • Windows or *nix users: install.packages("forensim", repos="http://r-forge.r-project.org")
  • MacOS users: install.packages("forensim", repos="http://r-forge.r-project.org", type = 'source')
