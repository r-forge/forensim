forensim download page




forensim package sources:
  • Source (Linux, Unix,  MacOS\Windows)
  • Windows binary
  • MacOS X binary
   
forensim development version is available from R-Forge's daily snapshots:
    • Windows or Linux users: install.packages("forensim", repos="http://r-forge.r-project.org") 
    • MacOS users: install.packages("forensim", repos="http://r-forge.r-project.org", type = 'source')
Please note that these are the DEVELOPPEMENT versions, preferably use the above sources
